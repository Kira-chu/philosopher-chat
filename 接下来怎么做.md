# ✅ 接下来怎么做 - 完成部署

## 第一步：确认 Root Directory（你已经找到了！）

我看到你已经输入了 `/backend`，但应该输入 `backend`（不要前面的斜杠）

1. **在 Root Directory 输入框中**
   - 如果显示 `/backend`，改成 `backend`（删除前面的斜杠）
   - 如果显示 `backend`，那就对了，不用改

2. **点击 "Save" 或 "Update" 按钮保存**

---

## 第二步：添加环境变量

1. **点击顶部的 "Variables"（环境变量）标签**
   - 在 "Settings" 标签旁边

2. **点击 "New Variable"（新建变量）按钮**

3. **添加第一个变量：**
   - **Name（名称）**：`DEEPSEEK_API_KEY`
   - **Value（值）**：`sk-27b719807adf432a977ace6fdfb58d59`
   - 点击 **"Add"** 或 **"Save"**

4. **再次点击 "New Variable"，添加第二个变量：**
   - **Name**：`DEEPSEEK_API_BASE`
   - **Value**：`https://api.deepseek.com`
   - 点击 **"Add"**

5. **再次点击 "New Variable"，添加第三个变量：**
   - **Name**：`DEEPSEEK_MODEL`
   - **Value**：`deepseek-chat`
   - 点击 **"Add"**

---

## 第三步：重新部署

添加环境变量后，Railway 会自动重新部署。如果没有自动部署：

1. **点击 "Deployments"（部署）标签**
2. **找到最新的部署记录**
3. **点击右侧的 "..."（三个点）菜单**
4. **选择 "Redeploy"（重新部署）**

或者：

1. **点击左侧的 "philosopher-chat" 服务卡片**
2. **点击 "Deploy" 按钮**

---

## 第四步：等待部署完成

1. **等待 2-3 分钟**
2. **查看部署状态**
   - 如果显示 "Active" 或 "Success"，说明部署成功！✅
   - 如果显示 "Failed"，点击 "View logs" 查看错误信息

---

## 第五步：获取后端 URL

部署成功后：

1. **点击 "Settings" 标签**
2. **找到 "Domains"（域名）部分**
3. **点击 "Generate Domain"（生成域名）**
4. **复制生成的 URL**（例如：`https://philosopher-chat-production.up.railway.app`）
5. **这个 URL 后面会用到！**

---
